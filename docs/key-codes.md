# key codes

reminder:
  - CSI: `ESC [`
  - SS3: `ESC O`

## arrows

```
up      CSI A
down    CSI B
right   CSI C
left    CSI D
home    CSI H  (alt: CSI 1~)
end     CSI F  (alt: CSI 4~)
ins     CSI 2~
del     CSI 3~
pgup    CSI 5~
pgdn    CSI 6~

S-up    CSI 1;2A
S-down  CSI 1;2B
S-right CSI 1;2C
S-left  CSI 1;2D

C-up    CSI 1;5A
C-down  CSI 1;5B
C-right CSI 1;5C
C-left  CSI 1;5D

S-C-up  CSI 1;6A
S-C-dn  CSI 1;6B
S-C-rt  CSI 1;6C
S-C-lt  CSI 1;6D
```
